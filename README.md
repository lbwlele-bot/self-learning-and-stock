# AI 协同学习与投资辅助

这个目录已经做过一次收束。

如果你不想看很多文件，先看这 3 个就够：

- `START_HERE.md`
- `ESSENTIALS.md`
- `memory/principles.md`

目录现在分成 4 层：

- `memory/`
  - 长期背景、原则、开放问题
- `docs/playbooks/`
  - 真正拿来用的检查表和工作流
- `docs/frameworks/`
  - 理解系统用的框架文件
- `docs/cases/`
  - 历史样本和示例

推荐用法：

1. 平时先用 `docs/playbooks/NIGHTLY_CHECKLIST.md`
2. 局势不快时用 `docs/playbooks/WEEKLY_CHECK.md`
3. 重大节点前用 `docs/playbooks/PRE_CATALYST_CHECKLIST.md`
4. 只有需要时再去翻 `docs/frameworks/` 和 `docs/cases/`
