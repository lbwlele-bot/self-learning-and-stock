# Self Learning And Stock

这是一个围绕`投资决策辅助 + 认知成长`搭建的本地知识系统。

它不是自动荐股工具，也不是单纯的学习笔记堆。
它的核心目标是把：

`逻辑基点 -> 工作假设 -> 投资翻译 / 执行`

这条链条长期维护成一个可复用、可迭代、可复盘的系统。

## 这套系统解决什么问题

- 帮助把宏观判断、主题理解和资产语言分层处理
- 帮助把“学习成长”嵌入“投资系统”前半段
- 帮助从逻辑出发，逐步发现值得研究的潜在承接者
- 帮助避免把热门共识股误当成预期差对象
- 帮助把观察、验证、仓位和动作收口成可执行规则

## 目录结构

- `memory/`
  长期记忆层：逻辑基点、原则、开放问题、阶段摘要
- `docs/playbooks/`
  执行层：观察、检查、动作收束
- `docs/frameworks/`
  理解层：发现路径、阶段交易、时间框架等
- `docs/cases/`
  历史样本与示例
- `templates/`
  会话模板、市场检查模板、潜在承接者研究模板

## 当前架构原则

- `主题优先导航`
- `三层骨架约束`
- `唯一主文件`
- `清晰优先，不以删除为目标`
