# 从这里启动

新对话默认先读：

1. `START_HERE.md`
2. `SYSTEM_INDEX.md`
3. `TOPIC_MAP.md`
4. `memory/logic-bases.md`
5. `memory/context.md`
6. `memory/principles.md`
7. `memory/open-questions.md`

如果只是继续讨论，通常到这里就够了。

## 默认约定

- 主导航：`主题优先`
- 底层骨架：`逻辑基点 -> 工作假设 -> 投资翻译 / 执行`
- 最高优先级：`区分事实 / 推断 / 猜测`
- 默认工作方式：`后台深搜核实，前台轻量收束`
- 新观点先判断归属，再决定写进哪里

## 新对话启动语

```text
先读取当前目录里的 START_HERE.md、SYSTEM_INDEX.md、TOPIC_MAP.md、memory/logic-bases.md、memory/context.md、memory/principles.md、memory/open-questions.md，然后按这套系统继续和我讨论，不要从零开始。
```
