# 个股前置信号栈

说明：

- 这份文件是：
  `公司层研究不只依赖年报/季报的前置信号 reference`
- 它服务于：
  - `docs/stocks/` 里的公司研究卡
  - `skills/company-card-builder/`
  - 一切“为什么这家公司值得持续观察”的讨论

一句话：
`个股层不能只看财报，也要看前置信号。`

## 先校正一个常见误区

个股研究时，不要默认一定能找到一个像：

- 黄金
- 美元
- 日元

那样相对完整、统一、稳定的“总逻辑”。

对个股来说，更常见的真实情况是：

`没有单一根基，只有逐步逼近真相的证据链。`

也就是说，个股研究更适合回答：

- 它靠什么赚钱
- 增量来自哪里
- 哪些信号说明逻辑在推进
- 哪些信号会打脸
- 这些证据，够不够支撑当前仓位级别

而不是强求自己必须先找到一个“一锤定音的大逻辑”。

## 这层回答什么

当你已经确定一家公司值得研究时，
接下来不要只问：

- 财报好不好
- 利润高不高

还要继续问：

1. 除了财报，哪些信号能更早说明主逻辑在推进
2. 哪些信号只是热闹，哪些信号更接近收入兑现
3. 哪些信息适合决定“继续观察”，哪些信息才足以升级到“考虑参与”

## 个股研究的最小证据链

默认至少拼出这 5 块：

1. `产品或业务竞争力`
2. `未来增量来源`
3. `前置信号`
4. `财务验证`
5. `反证条件`

这 5 块不要求一次性都完美，
但它们合起来，应该足以回答：

`现在这家公司，够不够支撑“继续观察 / 1% 试探 / 等进一步验证”中的哪一种动作。`

## 默认 5 层信号

### 1. 正式财务层

包括：

- 年报
- 季报
- 业绩预告
- 毛利率、现金流、存货、合同负债等质量指标

作用：

- 提供最终验证
- 校正“故事是否真的进了利润表”

局限：

- 最滞后
- 通常已经被市场提前交易了一部分

但注意：

- 个股异动时，不能只把财报层当作“慢信息”
- 盘后发布的 `季报 / 业绩预告 / 业绩快报 / 重大公告`
  往往就是次日最直接的定价触发器
- 对高关注个股，`是否刚发了新公告` 必须优先于一切历史研报和旧财报解读

### 2. 公司动态层

包括：

- 投资者关系活动记录
- 官网新闻
- 产品发布
- 开发者大会
- 合作公告
- 招聘方向

作用：

- 比财报更快地暴露公司重点方向
- 帮助判断管理层是否持续加码同一条路线

局限：

- 容易偏乐观
- 不能单独当成交逻辑

### 3. 产业链验证层

包括：

- tapeout / 样片 / 量产进度
- 产能 / 封测 / 交付
- 客户导入 / design-in
- 方案商反馈
- 竞品替代
- 场景是否真的上量

作用：

- 最容易形成预期差
- 最接近“主逻辑是不是在真实推进”

局限：

- 信息碎
- 难度高
- 要特别防止把传闻当事实

### 4. 行业与政策催化层

包括：

- 央视新闻 / 央视网
- 政府部门政策
- 展会
- 行业大会
- 标准制定
- 产业联盟动态

作用：

- 提供宏观到公司之间的桥梁
- 判断公司所处赛道是否在被持续强化

局限：

- 只能说明方向有风，不说明公司一定受益

### 5. 市场翻译层

包括：

- 股价位置
- 成交量
- 相对强弱
- 回撤深度
- 板块联动

作用：

- 帮助判断市场已经提前交易了多少
- 决定第一笔参与更适合在哪类位置出现

局限：

- 不告诉你真相
- 只能告诉你市场当前怎么定价

## 研究顺序

默认按下面顺序看：

1. 先用 `行业与政策催化层` 判断方向是否真的存在
2. 再用 `公司动态层` 看公司是不是持续押在这条线上
3. 再用 `产业链验证层` 看逻辑是不是正在走向兑现
4. 再用 `正式财务层` 校正故事是否已经进利润表
5. 最后用 `市场翻译层` 判断是否值得进入观察或轻仓参与

## 个股异动时的强制检查顺序

如果讨论的是：

- 为什么今天逆势
- 为什么突然大涨 / 大跌
- 现在能不能参与
- 明天市场会怎么交易

先不要直接讲逻辑，默认按下面顺序检查：

1. `最新公告`
   - 季报
   - 业绩预告
   - 业绩快报
   - 年报预约披露变更
   - 重大合作 / 订单 / 回购 / 激励
2. `公告时间`
   - 盘中
   - 盘后
   - 次日生效日期
3. `市场位置`
   - 个股前期是低位、横盘还是强趋势
4. `板块联动`
   - 是个股独立走强，还是整条板块一起走
5. `再解释`
   - 公告到底强化了什么
   - 市场是在兑现，还是在重估

未经过这一步，不把个股异动轻易归因为“资金偏好”或“市场认可”。

## 业绩事件的定价，不只看 PE

很多时候，市场对一份业绩的反应，不是单纯在回答：

- 现在 PE 贵不贵
- 这一季利润好不好看

更是在回答：

- 这份报告有没有明显 `surprise`
- 它强化的是老平台，还是新平台
- 它有没有把“故事”推进成“收入 / 量产 / 客户发布”
- 它会不会把市场的下一段预期抬高一档

所以对业绩事件，要分开看两件事：

1. `静态合理性`
   - 按当前利润和估值，股价大概处在哪个区间
2. `事件重估性`
   - 这份报告会不会让市场愿意提前交易更远一段的增长

第二项如果被低估，就会系统性低估市场反应。

## 适合升级观察的信号

更值得重视的，通常是：

- 新产品进入具体场景
- 客户导入更具体
- 设计导入 / 方案落地
- 量产和交付出现新进展
- 财务指标开始配合

## 不适合单独升级观察的信号

单独看到这些，先不要过度兴奋：

- 展会亮相
- 管理层“看好未来”
- 模糊的合作签约
- 行业热度大涨
- 板块联动上涨
