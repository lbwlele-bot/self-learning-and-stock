# 政策信号层

这是一层新增的观察层，用来补足当前系统只看“市场反应”的不足。

当前系统原本更擅长：

- 风险事件
- 情绪变化
- 市场价格反应

但对于政策驱动型行情，仅仅看上证、沪深300、Brent、美元指数，往往会偏慢。

所以新增一层：

- 政策信号层

## 1. 为什么需要这层

因为很多 A 股行情，先动的不是情绪，而是：

- 政策态度变了
- 政策力度超预期
- 政策工具第一次出现
- 多个部门同时发力

这些东西如果不先看清，等完全反映到价格里，往往已经走了一段。

## 2. 当前阶段先看哪些政策信号

不追求很多，先只看四类：

1. 货币宽松
2. 房地产支持
3. 资本市场支持
4. 高规格联合表态

## 3. 这些词是什么意思

### 货币宽松

典型动作：

- 降准
- 降息
- 逆回购利率下调

你可以先简单理解成：

- 让市场上的钱更便宜
- 让银行体系更宽松
- 让整体融资环境变得没那么紧

### 降准

意思：

- 降低银行必须交给央行冻结的准备金比例

你可以先把它理解成：

- 银行手上能动用的钱更多了
- 市场长期流动性变宽松

它更像：

- 放水
- 给银行体系加长期弹药

### 降息 / 7天逆回购利率下调

意思：

- 央行下调一个关键短期政策利率

你可以先简单理解成：

- 钱的“官方价格”下降了
- 后续会影响贷款利率、市场利率、融资成本

为什么 7 天逆回购重要：

- 它是一个很核心的短期政策利率锚
- 它一动，市场会认为后面一系列利率也可能往下走

它更像：

- 给市场一个“利率方向明确转松”的信号

### 存量房贷利率下调

意思：

- 已经背着房贷的人，后续可以按更低的利率还贷

你可以先理解成：

- 给居民减负
- 让居民少还一点利息
- 试图改善消费和地产链压力

它不是直接刺激股市的工具，但会影响：

- 居民现金流
- 地产预期
- 银行和消费相关板块预期

### 房地产支持

典型动作：

- 降首付
- 降存量房贷利率
- 放松限购限贷

这类信号的重要性在于：

- 地产在中国经济里权重很大
- 地产一动，市场会联想到经济、信用、消费、银行、地方财政

### 资本市场支持工具

典型动作：

- 支持中长期资金入市
- ETF 增持
- 股票回购增持专项工具
- 证券、基金、保险互换便利

这类信号的重要性在于：

- 它们不是泛泛表态，而是直接冲着股票市场来的
- 市场会理解成“政策开始主动呵护资本市场”

### 高规格联合表态

什么意思：

- 不是单个部门放风
- 而是多个关键部门一起发声，甚至在更高层会议中集中释放信号

这类信号通常意味着：

- 不是局部修修补补
- 而是政策态度本身出现了更明确的统一

## 4. 这层怎么接到当前系统里

以后可以把观察顺序改成：

1. 先看事件类型
   - 风险事件
   - 政策事件
   - 混合事件
2. 如果是政策事件，先看政策信号层
3. 再看市场变量层
   - 上证
   - 沪深300
   - Brent
   - 美元指数

## 5. 当前阶段怎么用

先不要把政策层做得很复杂。

只问三个问题：

1. 这次政策是小修小补，还是态度明显变化？
2. 这次政策是一条，还是多条同时出现？
3. 这次政策是嘴上表态，还是带着真工具来？

如果答案偏向：

- 态度变了
- 多条一起出
- 带真工具

那它就值得被系统认定为强政策信号。
