# 组合预算与角色分工

这层回答一个动作层已经越来越需要、但系统此前没有单独讲清的问题：

- 同样看好中国、看好 A 股，
- 到底应该优先买 `宽基 / ETF`，
- 还是优先加 `主题表达`，
- 还是直接提高 `高 conviction 个股` 权重？

一句话：

`先分清组合里每一笔钱在解决什么问题，再决定该把预算给谁。`

## 1. 这层回答什么

这层主要回答：

1. 组合里不同仓位分别承担什么角色
2. 当总判断偏向“整体中国资产值得参与”时，预算应先给宽基还是个股
3. 当你想吃整体上升收益、又想降低个股风险时，默认应该怎么分工
4. 什么情况下可以从 `宽基优先` 升级到 `主题 / 个股优先`

## 2. 这层不回答什么

这层不直接回答：

- 明天要不要下单
- 今天是不是加 1%
- 某只股票现在是不是买点
- 精确到小数点的固定仓位公式

这些分别属于：

- `../playbooks/ACTIONS.md`
- `TRADE_TYPE_CHECK.md`
- `TIME_HORIZONS.md`

## 3. 为什么现在需要这层

系统此前已经知道：

- `资产的角色先于价格表现`
- `现金、宽基、主题资产、对冲资产在组合中承担不同任务`

但在动作层里，
此前更擅长处理的是：

- 首次参与
- 事件验证
- 试探与撤回
- 分层吸纳

它还不够系统地回答：

- `整个组合里，应该先给谁预算`

尤其当当前目标不是“证明自己选股能力”，
而是：

- `参与中国 A 股整体修复或上升`
- `降低单一个股带来的判断误差`

这时组合层就必须单独长出来。

## 4. 当前先区分五种角色

### A. 等待权仓

更接近：

- `现金`
- `货币类低波动持有`

它解决的是：

- 保留未来选择权
- 给系统留出纠错空间
- 避免组合在错误时间点被迫满仓承担风险

### B. 宽基 Beta 仓

更接近：

- `沪深300 / 中证A500 / 上证50 / 全市场宽基 ETF` 一类广义指数暴露

它解决的是：

- 当你判断 `中国 / A股整体值得参与`
  时，
  先拿到整体市场收益
- 降低“方向对了、但个股选错了”的风险
- 把国家级、市场级判断翻成最朴素的权益表达

### C. 主题表达仓

更接近：

- 行业 ETF
- 板块 ETF
- 某条主线的中观表达

它解决的是：

- 当你不只看好 A 股整体，
  还看好其中某个方向的相对强势
- 在 `宽基` 和 `个股` 之间，
  提供一个中等难度、中等风险的表达层

### D. 高 conviction 个股仓

更接近：

- 已形成较完整证据链的少数个股

它解决的是：

- 当你对具体承接者、具体公司有研究优势时，
  争取超额收益

它的代价是：

- 个股错误
- 节奏错误
- 证据链不完整时的主观放大

### E. 对冲 / 安全仓

更接近：

- `黄金`
- 其他体系对冲类资产

它解决的是：

- 不是赚最多，
  而是在组合里处理：
  `地缘 / 信用 / 流动性 / 系统性不确定`

## 5. 预算分配的默认顺序

第一版先不做复杂公式，
只给预算顺序。

默认先按下面顺序分：

1. `先留等待权`
2. `再定对冲 / 安全仓`
3. `再定宽基 Beta 仓`
4. `然后才是主题表达仓`
5. `最后才提高高 conviction 个股仓`

这背后的意思不是：

- 个股永远不重要

而是：

- 除非你已经明确拥有比市场更好的对象识别能力，
  否则不该一上来就把最多预算给最难的那一层

## 6. 什么时候宽基应优先于个股

如果同时满足下面几条，
默认优先提高 `宽基 / ETF` 预算，
而不是先提高单一个股仓位：

1. 你更强的判断是：
   `中国 / A 股整体值得参与`
   而不是
   `我已经高度确认某一只个股会明显跑赢`
2. 你担心的主要风险是：
   `选错个股`
   而不是
   `判断错中国整体方向`
3. 市场风格分化明显，
   热点端已经较拥挤，
   但非热点板块仍存在承接空间
4. 你当前对赛道或个股的研究，
   还没有形成比 `宽基 Beta` 更高的确定性优势
5. 你的目标更接近：
   `先吃到中国资产整体修复收益`
   而不是
   `现在就追求个股超额收益最大化`

一句话：

`如果你更想先买中国，而不是先证明自己会选股，预算就先给宽基。`

## 7. 当前与你最相关的默认版本

如果当前目标是：

- `想提高中国 A 股整体参与度`
- `不想继续过度依赖单一个股`
- `愿意接受市场波动，但不想把结果过多押在选股上`

那第一版更适合用：

`等待权 / 安全仓 + 宽基核心 + 少量主题 + 更少量个股`

来搭组合，
而不是：

`大量分散个股 + 少量指数基金`

### 一个可用的第一版顺序

先做三层：

1. `底层`
   - 等待权
   - 对冲 / 安全仓

2. `核心权益层`
   - 宽基 / ETF

3. `增强层`
   - 主题表达
   - 高 conviction 个股

如果现在要提高中国 A 股参与度，
默认优先加的是：

- `核心权益层`

而不是：

- 先扩张更多单一个股

## 8. 风险预算里的第一版比例语言

这不是硬公式，
只是当前阶段的默认锚点。

这里必须先做一个重要区分：

- 下面的比例，
  说的是：
  `A股权益风险预算内部怎么分`
- 不是在说：
  `总资产里一定要拿这么多去买股票`

也就是说，
组合分两层：

### 第一层：总资产层

先回答：

- 多少放 `固收 / 现金 / 低波动仓`
- 多少放 `权益风险资产`

对偏稳、仍在实践中学习、暂不以高风险博取高收益为目标的版本，
完全可以默认：

- `50% 以上` 放在 `固收 / 现金 / 低波动资产`
- 剩余部分再进入 `权益风险预算`

### 第二层：权益风险预算层

只有在你已经决定：

- `这部分钱就是要参与 A股权益`

之后，
才去继续讨论：

- `宽基 / ETF`
- `主题表达`
- `高 conviction 个股`

之间怎么分。

所以如果你总资产里：

- `50%` 放固收
- `50%` 作为权益风险预算

那么下面写的：

- `50% - 70%` 给 `宽基 / ETF`

真正对应到 `总资产`，
其实只是：

- `25% - 35%` 的总资产

而不是我在建议你把总资产的 50% - 70% 全部压进宽基。

当你已经决定：

- 组合里要增加 `A 股权益风险预算`

那在这部分 `权益风险预算` 里，
第一版默认更偏向下面的结构：

### 版本 A：整体中国优先

适用：

- 主要目标是吃中国 A 股整体收益
- 个股优势暂时一般
- 风格偏稳，不希望把结果过多押在个股上

默认更接近：

- `50% - 70%` 给 `宽基 / ETF`
- `20% - 30%` 给 `主题表达`
- `0% - 20%` 给 `高 conviction 个股`

### 版本 B：主题有把握，但个股还在筛

适用：

- 对某些方向已有中观判断
- 但个股尚未形成强证据链

默认更接近：

- `40% - 60%` 给 `宽基 / ETF`
- `25% - 40%` 给 `主题表达`
- `10% - 20%` 给 `高 conviction 个股`

### 版本 C：已经形成少数强对象

适用：

- 少数个股已进入高 conviction 阶段
- 愿意承担更高个股波动

默认更接近：

- `35% - 50%` 给 `宽基 / ETF`
- `20% - 30%` 给 `主题表达`
- `20% - 35%` 给 `高 conviction 个股`

补充边界：

- 如果个股层还说不清主逻辑和反证，
  就不进入版本 C
- 在没有明确研究优势前，
  `高 conviction 个股` 不应反超 `宽基核心`

### 一个更贴近当前你的保守版本

如果你的总资产默认前提是：

- `至少 50%` 放在 `固收 / 现金 / 低波动资产`

那剩余 `50%` 左右的权益风险预算里，
更适合先按：

- `60%` 左右给 `宽基 / ETF`
- `25%` 左右给 `主题表达`
- `15%` 左右给 `高 conviction 个股`

去理解。

翻成总资产大致是：

- `30%` 左右总资产给 `宽基 / ETF`
- `12.5%` 左右总资产给 `主题表达`
- `7.5%` 左右总资产给 `高 conviction 个股`
- 其余 `50%` 仍留在 `固收 / 现金 / 低波动`

这就更接近：

- 先用指数基金吃中国 A 股整体收益
- 再用少量主题和个股做增强
- 同时不让组合一下子变成高波动权益组合

## 9. 宽基与主题怎么选

如果当前主要命题是：

- `中国经济和 A 股整体修复`

优先看：

- `宽基`

如果当前主要命题是：

- `A 股内部不是全面牛，而是局部承接`

那就在宽基之外再补：

- `主题表达`

也就是说：

- `宽基` 解决的是“先买中国”
- `主题` 解决的是“再买结构”
- `个股` 解决的是“最后买超额收益”

## 10. 什么时候从宽基升级到主题或个股

只有当下面信号逐步增加时，
才把预算从 `宽基核心` 往上升级：

1. 不是只有科创或少数热点在涨，
   而是出现更广泛的承接
2. 主题的受益路径，
   已经从政策或叙事走到预算、订单、盈利传导
3. 你能说清：
   `这条主线为什么会比大盘更强`
4. 对具体个股，
   已经形成：
   `产品竞争力 + 增量来源 + 前置信号 + 财务验证 + 反证条件`
   的证据链

如果以上 4 条仍不清楚，
就优先待在：

- `宽基 / ETF`
- `主题 ETF`

而不是过早往个股集中。

## 11. 当前最重要的反方提醒

### 错误一：把分散个股误当成分散风险

买很多个股，
不等于真的分散。

如果它们本质都暴露在：

- 同一风格
- 同一主题
- 同一风险偏好

那只是：

- `形式分散，实质集中`

### 错误二：把宽基当成“没观点”

宽基不是放弃判断。

它是在承认：

- 你对“中国整体”比对“哪只最强”更有把握

### 错误三：太早把预算给最难的一层

高 conviction 个股，
应该来自：

- 研究优势
- 证据积累
- 持续验证

而不是来自：

- 觉得指数涨太慢
- 觉得个股更刺激
- 害怕错过更大弹性

## 12. 和现有动作规则的关系

这层先回答：

- `预算先给谁`

然后才进入：

- `ACTIONS.md`

也就是说：

- 组合层先决定：
  `宽基 / 主题 / 个股`
  各自预算地位
- 动作层再决定：
  `继续观察 / 1% / 分层吸纳 / 撤回`

## 13. 当前一句话版本

`当你的更强判断是“中国值得参与”，而不是“我已经高度确认哪只个股会赢”时，组合预算应先给宽基，再给主题，最后才给个股。`
