# 阶段交易何时开始失效

这层回答一个和“阶段性成立”同样重要的问题：

- 一段阶段行情，什么时候开始不该再按原来的逻辑继续拿？

## 1. 先说核心

阶段交易不是永远成立。

它一旦开始失效，最常见的问题不是：

- 方向突然完全反了

而是：

- 原来支撑这段行情的那几根柱子，开始松了

## 2. 阶段交易最常见的三根柱子

一段阶段行情通常靠这三样支撑：

1. 持续性
2. 扩散性
3. 承接力

所以失效也往往从这三样里开始出现裂缝。

## 3. 持续性失效

更接近这种状态时，要提高警惕：

- 后续政策、事件、资金不再继续跟
- 原本的主线开始失去新增催化
- 市场从“继续强化预期”变成“再也讲不出新东西”

一句话：

- 原来的故事还在，但已经没有新燃料了

## 4. 扩散性失效

更接近这种状态时，要提高警惕：

- 行情重新缩回少数方向
- 原来参与的更多板块开始掉队
- 只剩局部强势股在撑，整体赚钱效应变差

一句话：

- 这段行情不再往外扩，而是开始收缩

## 5. 承接力失效

更接近这种状态时，要提高警惕：

- 回调后接不住
- 低点开始明显下移
- 再次回落时，市场比前几次更脆弱

一句话：

- 一旦掉下来，就没有以前那么多人愿意接了

## 6. 现实校验开始占上风

很多阶段行情到后面，不是逻辑突然消失，而是：

- 现实开始检查它

常见表现：

- 预期打得很高
- 后续细节不够
- 政策力度没有继续超预期
- 盈利和基本面跟不上

这时就不能再只靠“前面那次启动很强”来解释后面的持有理由。

## 7. 当前最有用的判断方法

当你想继续拿一段阶段行情时，先问：

1. 现在还有没有新催化？
2. 行情是在扩散，还是在收缩？
3. 回调后还有承接吗？
4. 市场开始在交易新预期，还是开始做现实校验？

如果这几条里，大部分答案都偏向坏的一边，
就说明阶段交易开始失效。

## 8. 当前一句话总结

阶段交易失效，不一定是因为方向完全错了，而更常见的是：

- 这段行情赖以成立的持续性、扩散性和承接力，开始一起变弱了。
