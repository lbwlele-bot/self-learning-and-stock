# 交易类型检查

这层用来回答一个很实际的问题：

- 我现在到底在做什么类型的判断？

很多亏损，不是因为方向错了，而是因为：

- 本来做的是事件交易
- 却拿成长周期配置去做

或者反过来。

## 1. 三种常见类型

### A. 事件交易

更像在做：

- 某个突发事件
- 某次政策落地
- 某个短期预期差

典型特征：

- 触发因素很明确
- 市场会快速反应
- 重点在几天到一两周
- 最关心的是预期重定价和情绪扩散

你更可能在问：

- 这件事会不会让市场快速改看法？
- 第一波资金会去哪里？
- 这是不是一次短期重定价机会？

### B. 阶段交易

更像在做：

- 一段政策周期
- 一段风险偏好修复
- 一段板块或风格扩散

典型特征：

- 不是只有第一天
- 会看几周到几个月
- 重点在持续性、扩散性、回撤后的承接力

你更可能在问：

- 这次变化是不是会持续一段时间？
- 政策和资金会不会继续跟？
- 市场是脉冲，还是阶段性行情？

### C. 长周期判断

更像在做：

- 宏观周期
- 盈利周期
- 估值中枢变化

典型特征：

- 关注时间更长
- 不靠单次事件立论
- 更依赖基本面、盈利、政策传导

你更可能在问：

- 经济和盈利是不是进入新阶段？
- 这轮行情有没有长期基础？
- 未来几个月甚至更长，值不值得配置？

## 2. 快速自检问题

如果你在想的是：

- “这几天会不会快速重定价？”

那更像 `事件交易`。

如果你在想的是：

- “这一轮会不会走成一个阶段行情？”

那更像 `阶段交易`。

如果你在想的是：

- “这是不是长期趋势反转？”

那更像 `长周期判断`。

## 3. 当前最容易犯的错

### 错误一

- 用事件交易的逻辑进场
- 却用长周期的想象去硬拿

### 错误二

- 明明只是阶段修复
- 却要求它立刻证明长期基本面已经完全改善

### 错误三

- 明明做的是长周期
- 却被短期波动和情绪高潮带着跑

## 4. 当前建议的使用方式

每次判断前，先回答一句：

- 我现在是在做：
  - 事件交易
  - 阶段交易
  - 长周期判断

然后再问：

- 当前信号和动作，是不是匹配这个时间尺度？

## 5. 和当前系统的关系

### 事件交易更依赖

- 政策信号层
- 领先信号层
- 方向确认

### 阶段交易更依赖

- 持续性确认
- 情绪扩散
- 现实校验

### 长周期判断更依赖

- 基本面
- 盈利趋势
- 长期政策传导

## 6. 当前一句话总结

先分清自己在做哪一类，再谈该不该进、该不该拿。
