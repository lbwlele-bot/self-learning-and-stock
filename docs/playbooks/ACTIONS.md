# 第一版动作规则

这份文件只负责动作规则。

它是：
`执行规则的唯一主文件`

它位于方法层，不替代 skill 入口。
也不重复系统级原则，不替代逻辑基点、工作假设和投资翻译层。

对应的投资翻译主文件是：
`../frameworks/INVESTMENT_TRANSLATION.md`

当前第一版把动作拆成两段：

- `进入规则`
- `退出规则`

## 当前进入阶段只保留五种动作

1. `继续观察`
2. `小仓位试探`
3. `试探后再加一点`
4. `试探后撤回`
5. `分层吸纳`

## 1. 继续观察

适用：

- 最坏预期仍在继续强化
- 新增利空仍能持续把价格推向更差位置
- 你当前更多是感觉，而不是结构化判断

## 2. 小仓位试探

适用：

- 利空仍在，但市场开始对更坏情景钝化
- 主变量和复核变量不再继续同步恶化
- 这笔动作主要是验证，不是立刻证明自己对

## 3. 试探后再加一点

适用：

- 第一次试探后，市场继续给出钝化或修复反馈
- 没有新的事实迅速推翻原判断
- 你看到的是“环境继续配合”，而不是“自己更有信心”

## 4. 试探后撤回

适用：

- 新增利空再次把主变量推向更差位置
- 复核变量明显转弱
- 试探后市场迅速证明原判断不成立

## 5. 分层吸纳

这条规则是为了解决另一类常见问题：

- 不是所有机会都适合“等验证后再追”
- 有些对象本来就更适合在分歧、回撤、流动性错杀里慢慢吸纳
- 但如果没有边界，“越跌越买”很容易滑成情绪摊平

所以这里说的 `分层吸纳`，
不是“跌了就补”，
而是：
`只对少数适合长期建仓的对象，在主逻辑未坏前提下，用预设预算和分层位置去承接波动。`

### 只适用于什么对象

- `宽基 / ETF`
- `资产配置型标的`
- `已经形成高 conviction 的核心资产`
- `主逻辑清楚、证据链较完整、愿意中期持有的少数个股`

### 不适用于什么对象

- `事件票`
- `纯题材票`
- `靠单一 headline 驱动的脉冲标的`
- `自己没有明显研究优势、却只是觉得跌多了的热门股`

### 适用前提

只有同时满足下面 5 条，才允许从“试探”切到“分层吸纳”：

1. 你能说清：
   `这不是我在摊平亏损，而是在按计划建仓`
2. 对象本身属于：
   `适合中期持有 / 可反复观察 / 可被证据链验证`
3. 下跌更像：
   `流动性、风格、情绪或板块降温`
   而不是
   `基本面被重新定价`
4. 你已经提前设好：
   `总预算 / 分层比例 / 停止补的条件`
5. 你愿意接受：
   `它可能继续跌，而自己只能慢，不可以急`

如果这 5 条里有一条说不清：

- 不进入分层吸纳
- 回到 `继续观察 / 小仓位试探`

### 动作含义

- 这不是抄底机制
- 这不是均值回归迷信
- 这不是“我比市场聪明”

它只是：

- 防止高 conviction 对象在波动里始终没有建立仓位
- 允许把“长期看好”翻成 `慢吸纳`
- 用流程约束自己，不让“越跌越补”失控

一句话：
`分层吸纳不是赌反弹，而是给高 conviction 对象一个有边界的逆向建仓流程。`

### 默认动作方式

默认按三层处理：

- `第一层`：建立底仓
- `第二层`：回撤后再吸纳一层
- `第三层`：只有在主逻辑未坏、且位置继续给机会时才补齐

默认要求：

- 每一层都提前写明，而不是临场起意
- 后一层不能明显大于前一层
- 任何一层执行后，都要重新检查反证条件

### 和 1% 判断参与仓的区别

- `1% 判断参与仓`
  更适合：方向判断已成形，但时间点还不完全确定
- `分层吸纳`
  更适合：对象本身适合中期建仓，且你明确接受它不是一次验证完就结束的动作

也就是说：

- `1%` 重点在：`先建立连接`
- `分层吸纳` 重点在：`按预算慢慢承接`

### 默认预算边界

这不是固定仓位公式，
但默认要求至少满足下面三条：

- `先设总预算，再谈每层买多少`
- `单层只是总预算的一部分，不允许第一层就把预算打满`
- `预算用完前，必须还保留停止补的权利`

### 停止吸纳的条件

如果后面出现下面任一类，就停止继续补：

1. `主逻辑被证伪`
   - 原来支撑长期判断的事实开始反转

2. `跌因改变`
   - 原本像情绪或流动性，现在越来越像基本面问题

3. `对象弱于应有状态`
   - 同类中持续明显更弱
   - 板块修复时它不修
   - 利好出现时它也没有承接

4. `动作开始情绪化`
   - 你补仓的主要理由开始变成：
     - `已经跌这么多了`
     - `我不信还能再跌`
     - `摊低成本以后比较好受`

### 一句话版本

`只有当对象适合中期建仓、跌因更像错杀而不是坏掉、并且总预算和停止条件都先写清时，才允许把“越跌越买”翻译成“分层吸纳”。`

## 当前边界

- 这不是重仓系统
- 这不是高频切换系统
- 这不是完整仓位公式
- 组合预算与角色分工，优先回到 `../frameworks/PORTFOLIO_ROLE_BUDGET.md`
- 估值带如何翻成动作带，优先回到 `../frameworks/VALUATION_ACTION_BANDS.md`

它只负责：
`把观察连接到最小动作。`

## 补充规则：70 分判断的 1% 动作

这条规则只解决一种常见问题：

- 方向判断其实已经成形
- 但因为时间点不够舒服
- 总想“再看一天”
- 最后连最小试探都没有做

它不替代原有的：

- `继续观察`
- `小仓位试探`
- `试探后再加一点`
- `试探后撤回`

它只是补充：
`什么时候允许自己先做极小动作，去买自己的判断。`

### 适用前提

只有同时满足下面三条，才允许触发：

1. 你已经能用一句话说清：
   `我押的是什么`
2. 你已经能用一句话说清：
   `为什么更可能是现在，而不是以后`
3. 你已经能用一句话说清：
   `什么事实会证明我错`

如果三条里有一条说不清：

- 不触发
- 继续观察

### 动作含义

- 这笔动作不是为了赚很多
- 它是为了：
  - 防止“方向对但完全没参与”
  - 让判断接受市场验证
  - 让后续加仓有起点

一句话：
`1% 不是收益最大化仓，而是判断参与仓。`

### 默认动作大小

在当前阶段，默认只允许：

- `总资金的 1%`

它只适合：

- 判断已成形
- 但市场还没完全确认
- 你愿意为时间点不确定付出很小的试错成本

### 触发方式

如果到当日临近收盘时，你仍然认为：

- 方向判断没有被当天事实推翻
- 三句判断仍然说得清

那么允许：

- 在收盘前或当日最后观察窗口内
- 做 `1% 判断参与仓`

目的不是抢当天波动，
而是避免把“再看一天”无限拖成“始终不动手”。

### 和后续动作的关系

- 这 1% 不自动升级为加仓信号
- 后续是否加到 `2%-4%`
  仍然取决于：
  - 市场是否继续配合
  - 主变量和复核变量是否继续验证原判断
  - 有没有新的事实推翻原判断

### 当前边界

- 这条规则不适用于重仓判断
- 不适用于情绪冲动时临时找理由
- 不适用于三句判断说不清的情况
- 不允许因为“后悔踏空”而事后补造触发条件

### 一句话版本

`当方向判断已成形、但时间点仍不确定时，允许用 1% 去买判断，而不是要求自己等到完全舒服才行动。`

### 额外提醒

`1% 判断参与仓` 和 `分层吸纳` 不能混用。

如果你说不清当前是在：

- `买一个方向判断`
还是在
- `吸纳一个高 conviction 对象`

那默认先按更保守的：

- `1% 判断参与仓`

而不是直接进入分层吸纳。

## 退出规则：从持有到撤回

这部分只解决一个此前系统没有单独长出来的问题：

- 仓位买完以后，后面到底该怎么管

它不替代前面的进入规则，
而是回答：

- `什么时候继续拿`
- `什么时候先不再加`
- `什么时候可以收一点`
- `什么时候该降级`
- `什么时候该直接撤回`

第一版先只服务于：

- `当前已经真实买入并进入管理状态的仓位`

包括：

- `黄金`
- `中海油服`
- `国电南瑞`
- `长江电力`

它当前不回答：

- 固定百分比止盈
- 固定百分比止损
- 宽基 / ETF 的完整退出规则
- 对所有资产类别一把尺子量到底

### 先按这个顺序判断

以后讨论已有仓位，先不急着问“要不要卖”，
先固定按下面顺序走：

1. `逻辑有没有坏`
2. `角色有没有变`
3. `市场是不是已经先兑现太多`

这样做的目的，是避免把下面三件事混成一件：

- `被证伪`
- `不再适合加仓`
- `涨多了先收一点`

### 五种标准退出动作

以后已有仓位的后续管理，默认只用下面五种动作表达：

1. `继续持有`
2. `冻结加仓`
3. `兑现一部分`
4. `降级持有`
5. `全部撤回`

### 1. 继续持有

适用：

- 主逻辑仍在
- 当前角色没有变化
- 市场也没有明显把预期提前打满

它的含义是：

- 这笔仓位继续承担原任务
- 暂时不需要因为短期不够强就自我否定

### 2. 冻结加仓

适用：

- 主逻辑还没坏
- 但新增验证不足
- 相对强弱开始变差
- 或当前位置不适合继续抬仓

它的含义是：

- 先不扩大暴露
- 让市场继续证明
- 但也不把“暂时不配合”直接等同于“应该卖出”

### 3. 兑现一部分

适用：

- 主逻辑还在
- 角色没有根本变化
- 但价格、情绪或预期已经明显跑在事实前面

它的含义是：

- 不是看空
- 而是承认：
  `市场可能已经先走了你原本想等事实兑现的那一段`

### 4. 降级持有

适用：

- 主逻辑未必被证伪
- 但它在组合里的任务已经变化
- 或仓位涨出来后，已经不再适合保留原级别角色

它的含义是：

- 从 `主表达`
  降回
  `辅助表达 / 观察仓 / 稳定器 / 保留入口`

这里也顺带回答：

- `什么时候做再平衡`

第一版里，
`再平衡` 不单列成新动作，
而是优先落在：

- `兑现一部分`
或
- `降级持有`

### 5. 全部撤回

适用：

- 主逻辑被证伪
- 跌因已经从情绪或节奏问题，变成基本面或主假设问题
- 市场持续证明原判断不成立

它的含义是：

- 不是“这票让我不舒服”
- 而是：
  `继续持有已经不再符合原有规则`

### 一个最小判断表

如果更像下面这些情况，默认先对应到：

- `逻辑没坏 + 角色没变 + 预期没打满` -> `继续持有`
- `逻辑没坏 + 角色没变 + 只是验证停滞` -> `冻结加仓`
- `逻辑没坏 + 角色没变 + 市场先打满预期` -> `兑现一部分`
- `逻辑没坏 + 角色变了` -> `降级持有`
- `逻辑坏了` -> `全部撤回`

### 不同角色，不用同一把尺子

这一步非常重要。

第一版不按“涨了多少 / 跌了多少”机械处理，
而是先看它在组合里承担什么任务。

#### A. 黄金

黄金当前更接近：

- `体系对冲`
- `战略安全垫`
- `长期风险入口`

所以先问的不是：

- `今天够不够强`

而是：

- `它是否仍值得保留对冲入口`

当前默认理解：

- 风险资产修复时，黄金不够强
  不自动等于该卖
- 只有当：
  - `美元和利率重新形成持续压制`
  - `地缘 / 信用 / 货币风险边际明显缓和`
  - `黄金修复持续偏弱，连对冲价值都在钝化`
  才开始讨论：
  `冻结更高暴露 / 降级 / 撤回`

#### B. 中海油服

中海油服当前更接近：

- `事件表达转研究观察仓`

所以核心不再是问：

- `霍尔木兹是不是非得长期封锁`

而是问：

- `它能不能从地缘脉冲切到海上油服景气或公司层逻辑`

当前默认理解：

- 事件溢价弱化、但公司没有立刻走坏：
  优先看 `冻结加仓`
- 如果事件溢价消退后，它持续弱于同类，
  且自己也说不清为什么还值得持有：
  优先看 `全部撤回`

#### C. 国电南瑞

国电南瑞当前更接近：

- `第二波承接`
- `电网进攻表达`
- `分两腿执行的计划仓`

所以关键不只是看涨跌，
而是看：

- `市场修复是否继续扩散`
- `电网链是否形成持续承接`
- `订单 / 预算 / 兑现预期有没有继续支撑`
- `它是否持续弱于同方向更优对象`

当前默认理解：

- 扩散还没证伪，但也没继续验证：
  优先看 `冻结加仓`
- 扩散成立、对象继续配合：
  允许 `继续持有`
- 扩散被证伪、对象持续掉队：
  进入 `降级持有` 或 `全部撤回`

#### D. 长江电力

长江电力当前更接近：

- `权益稳定器`
- `波动缓冲层`
- `高现金流底仓`

所以先问的不是：

- `它为什么不领涨`

而是：

- `它还像不像一个稳定器`

当前默认理解：

- 不够强，不自动等于失败
- 只有当：
  - `稳定器属性减弱`
  - `异常弱势`
  - `来水 / 发电 / 分红 / 现金流逻辑明显破坏`
  - `仓位涨出来后已经超出原角色`
  才讨论：
  `冻结继续加仓 / 兑现一部分 / 降级持有`

### 当前边界

- 这不是高频做 T 规则
- 这不是机械止盈止损模板
- 这不是宽基 / ETF 的完整退出手册
- 这不是用一个百分比处理所有角色

它当前只负责：

- `让已有仓位的管理语言统一起来`

## 补充规则：亏损分类

这条规则只解决一种常见心理问题：

- 把所有亏损都当成同一种失败
- 结果既不敢开仓，也很难拿住浮盈
- 最后为了避免账面亏损，反复制造机会亏损

### 1. 合理亏损

定义：

- 按规则进行的小仓位试探
- 后续被市场或事实证伪
- 在原有边界内接受的小额回撤或退出

它不等于失败。

更合适的理解是：

- `试错成本`
- `信息费`
- `系统运行成本`

一句话：
`合理亏损是允许存在的，不应用情绪放大。`

### 2. 违规亏损

定义：

- 判断没有成形就临时找理由上
- 明明被证伪还不撤
- 因为情绪、后悔或不甘心而扩大仓位
- 把试探仓做成冲动仓

这类亏损才是需要重点警惕的。

一句话：
`真正危险的不是亏，而是破坏规则后再亏。`

### 3. 机会亏损

定义：

- 方向判断大体正确
- 但因为过度等待、过度求稳、总想“再看一天”
- 连最小参与仓都没有建立
- 最后完全错过本应属于自己的那一段

它虽然不出现在账面上，
但会显著伤害判断、自信和执行一致性。

一句话：
`为了避免合理亏损，反复制造机会亏损，也是一种亏。`

### 当前使用方式

以后每次复盘，不只问：

- `我亏没亏`

还要多问两句：

- `这次属于合理亏损、违规亏损，还是机会亏损`
- `我是在保护自己，还是只是在逃避不舒服`

### 当前目标

不是追求：

- `永远不亏`

而是逐步做到：

- 接受合理亏损
- 避免违规亏损
- 减少机会亏损

### 一句话版本

`账面不亏，不等于真的没亏；为了避免小亏而长期错过本应参与的机会，也是一种需要被记录的亏损。`

## 临场提醒：不要把逃避包装成理性

当你已经明显感觉到自己在反复犹豫时，先不问更多问题，
先问这一句：

`这次不动，是在保护规则，还是在逃避不舒服？`

### 防自我开脱检查

为了避免把“逃避”解释成“我在保护规则”，
再多过下面这 3 句：

1. 如果今天不做这 1%，我说得出是哪一条已写明规则阻止了我吗？
2. 如果说不出具体规则，我是不是只是还想再等一个让我更舒服的信号？
