# 第一版动作规则

这份文件只负责动作规则，属于执行层。
不重复系统级原则，不替代逻辑基点与工作假设澄清。

## 当前只保留四种动作

1. `继续观察`
2. `小仓位试探`
3. `试探后再加一点`
4. `试探后撤回`

## 1. 继续观察

适用：

- 最坏预期仍在继续强化
- 新增利空仍能持续把价格推向更差位置
- 你当前更多是感觉，而不是结构化判断

## 2. 小仓位试探

适用：

- 利空仍在，但市场开始对更坏情景钝化
- 主变量和复核变量不再继续同步恶化
- 这笔动作主要是验证，不是立刻证明自己对

## 3. 试探后再加一点

适用：

- 第一次试探后，市场继续给出钝化或修复反馈
- 没有新的事实迅速推翻原判断
- 你看到的是“环境继续配合”，而不是“自己更有信心”

## 4. 试探后撤回

适用：

- 新增利空再次把主变量推向更差位置
- 复核变量明显转弱
- 试探后市场迅速证明原判断不成立

## 当前边界

- 这不是重仓系统
- 这不是高频切换系统
- 这不是完整仓位公式

它只负责：
`把观察连接到最小动作。`
