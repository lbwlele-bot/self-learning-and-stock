# 日常更新触发词

如果要让我接管“当天信息抓取 + 模板填写 + 初步判断”，你以后直接发下面这段就行：

```text
先读取当前目录里的 START_HERE.md、README_TRADING_LOOP.md、memory/ 和最近两条 sessions 记录。
然后基于最新信息，完成今天的市场观察：
1. 抓取当天与当前主题相关的重要新闻和市场数据
2. 区分背景风险、当天催化、情绪叙事
3. 按模板填写今天的观察记录
4. 给出一个简短结论：
   - 风险继续升级
   - 风险仍在，但市场开始钝化
   - 情绪修复，但还不稳定
5. 如有必要，更新原则库和开放问题
```

更短版本：

```text
按当前目录里的交易观察框架，帮我做今天的市场更新。
```

适用场景：

- 每天收盘后更新
- 出现重大国际事件后更新
- 想快速判断“市场现在在交易什么”时使用
