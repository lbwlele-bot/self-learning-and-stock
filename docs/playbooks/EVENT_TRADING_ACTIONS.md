# 事件交易的最小动作规则

这层只针对：

- 你已经判断当前更像一笔 `事件交易`

它不回答所有问题，只回答三个最关键的问题：

1. 什么时候可以开始试？
2. 什么时候已经偏晚？
3. 什么时候不能再拿事件逻辑硬扛？

## 1. 什么时候可以开始试

更接近可以开始试的时候，通常满足下面几类特征：

- 领先信号已经出现
  - 高规格事件/政策
  - 工具比预期强
  - 市场第一批直接受益方向开始明显响应
- 方向开始被确认
  - 不是只有一条消息
  - 而是市场开始真正重新定价

这时更适合做的，不是重仓，而是：

- 小仓位试探

一句话：

- 当你看到方向开始明确，但持续性还没完全确认时，是事件交易最典型的试探区间

## 2. 什么时候已经偏晚

更接近偏晚的时候，通常会出现：

- 指数已经明显大涨
- 直接受益方向已经连续强势
- 情绪和舆论都已经很热
- 市场开始讲越来越大的故事

这时不是说一定不能涨了，而是要意识到：

- 你买进去时，吃到的已经不再是最初的方向确认
- 更可能是在接情绪扩张和预期打满之后的部分

一句话：

- 当“大家都看懂了”时，通常不是没有机会，而是事件交易最便宜的部分已经过去了

## 3. 什么时候不能再拿事件逻辑硬扛

如果出现下面这些情况，就不应该继续只靠最初事件逻辑硬拿：

- 现实校验已经开始
- 后续细节没有跟上最初预期
- 价格已经明显从情绪高潮回落
- 你自己的理由开始从“事件驱动”偷偷变成“长远总会好”

一句话：

- 当市场已经开始检查兑现度时，就不能再只靠最初的事件冲击来解释一切

## 4. 当前最小动作顺序

### 第一步：试探

在方向确认阶段，如果领先信号够强，可以：

- 小仓位试探

### 第二步：观察升级还是降级

后面重点看：

- 是不是进入持续性确认
- 还是很快被现实打脸

### 第三步：不要在逻辑变质后硬拿

如果已经从事件交易进入：

- 情绪高潮
- 现实校验

那就必须重新回答：

- 我现在还是在做事件交易吗？

如果不是，就不能继续沿用最初的动作逻辑。

## 5. 当前一句话总结

事件交易最合理的动作，不是等一切确认后再追，也不是靠第一天逻辑一路硬扛，而是在方向刚确认时试探，在逻辑变质后及时重新分类。
