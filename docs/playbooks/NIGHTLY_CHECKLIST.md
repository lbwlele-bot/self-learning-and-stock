# 晚间检查清单

目的：

- 不做短线盯盘
- 只在晚上用几分钟判断：明天值不值得多留一点注意力

## 一、先看明天是不是普通日子

只问 3 个问题：

1. 明天有没有已知的重要会议、发布会或政策窗口？
2. 最近两三天，官方口径有没有明显在往某个方向集中？
3. 最直接受益的方向，有没有提前异动？

## 二、快速分级

### `普通观察`

- 没有明确日程窗口
- 没有明显政策预热
- 没有直接受益方向异动

### `提高注意力`

- 三个问题里，有 1 到 2 个开始成立
- 说明明天可能不是完全普通的一天

### `建议预留时间`

- 已知高规格窗口明确存在
- 再叠加预热或异动
- 说明明天值得你中午或收盘前专门看一下

## 三、你实际要做什么

如果是：

- `普通观察`
  - 不额外安排时间
- `提高注意力`
  - 明天中午顺手看一下
- `建议预留时间`
  - 明天中午或收盘前专门抽几分钟

## 四、触发词

晚上直接发一句：

```text
按晚间检查清单帮我看一下，明天是普通观察、提高注意力，还是建议预留时间。
```

## 五、边界

- 这不是预测明天涨跌
- 只是判断明天值不值得你多留一点注意力
