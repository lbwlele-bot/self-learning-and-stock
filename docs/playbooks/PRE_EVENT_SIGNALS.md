# 事前信号层

这层回答一个更早的问题：

- 在重大事件真正落地前，有没有机会先提高警觉？

答案是：

- 有时候有，而且很重要
- 但不是所有事件都有

所以这层的目标不是提前预测所有大事，而是：

- 尽量提前识别那些“值得预留注意力”的时点

## 1. 先分两类事件

### A. 可预热事件

这类事件通常更可能提前留下线索，例如：

- 国新办发布会
- 政治局会议
- 国务院重要会议
- 央行/证监会/金融监管总局的重要安排
- 财政、地产、资本市场政策窗口

特点：

- 往往有时间安排
- 有前期舆论铺垫
- 有政策吹风或市场预期

### B. 突发事件

例如：

- 战争突然升级
- 制裁落地
- 黑天鹅事故

特点：

- 很难提前精准知道
- 更多依赖即时预警，而不是事前准备

## 2. 政策类事件的常见事前信号

### 1. 日程信号

比如：

- 已公布的国新办发布会
- 已知的重要会议时间窗口
- 假期前后、月末、季末、重要宏观节点

### 2. 舆论预热信号

比如：

- 官方媒体开始密集谈稳增长、稳市场、提振信心
- 监管部门频繁释放相近口径
- 市场开始反复讨论某类政策窗口

### 3. 市场先动信号

比如：

- 券商、多元金融、地产链提前异动
- 指数没大动，但某些直接受益方向先悄悄走强
- 成交和风险偏好有异常变化

## 3. 当前最实用的事前问题

以后遇到政策窗口前，可以先问三句：

1. 这几天有没有已知的重要会议/发布会窗口？
2. 官方口径是不是在明显往某个方向集中？
3. 最直接受益的方向有没有提前异动？

如果三者里有两条明显成立，就值得把注意力等级调高。

## 4. 当前一句话总结

真正可行的不是“提前知道一切”，而是：

- 提前知道哪些日子和哪些线索值得你多留一点注意力
