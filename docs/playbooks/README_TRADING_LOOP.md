# 最小交易观察闭环

仅在`逻辑基点与工作假设已相对清楚`时使用。
这份文件属于执行层，不负责替代基点澄清和假设生成。

当前版本先回答四个问题：

1. 今天发生了什么新增信息？
2. 上证综指和 Brent 原油怎么反应？
3. 市场是在继续交易最坏预期，还是开始对最坏预期钝化？
4. 如果要参与，当前更适合先看哪类资产，而不是默认去买大盘？

对政策驱动型行情，额外补看：

5. 今天有没有出现强政策信号？

## 当前最小闭环

1. 抓新增信息
2. 做主变量和复核变量观察
3. 判断市场在交易事实还是预期变化
4. 判断最坏预期是在强化还是开始钝化
5. 再用 `ACTIONS.md` 收口到动作

## 当前主变量

- `上证综指`
- `Brent 原油`

## 当前复核变量

- `沪深300`
- `美元指数`

## 当前原则

- 不用单条新闻直接下结论
- 先判断环境，再判断风险暴露
- 先判断风险暴露，再判断具体载体
