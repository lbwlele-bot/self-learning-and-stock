# 长期观察模板

适用场景：

- 你对一家公司或一个方向有长期兴趣
- 但当前还没有清晰买点
- 你不想空谈“长期看好”，而是想持续跟踪它是不是在被验证

## 1. 为什么进入观察名单

- 我为什么会长期看它？
- 关联逻辑基点是什么？
- 这个判断来自：
  - 产业体感
  - 财报
  - 产品竞争力
  - 客户粘性
  - 其他：

## 2. 我押的核心逻辑

- 我押的不是“股价会涨”，而是：
- 这家公司未来最可能变强的地方是什么？
- 当前工作假设是什么？

## 3. 前置信号

- 哪些变化会让我觉得它在变强：
- 哪些变化会让我觉得它在变弱：

## 4. 后验验证

- 后面我应该看什么来验证自己不是在自嗨：
  - 收入
  - 利润
  - 毛利率
  - 订单 / 客户
  - 新产品
  - 其他：

## 5. 市场预期

- 这个逻辑现在是：
  - 没什么人看
  - 开始有人看
  - 已经很热

## 6. 当前定位

- 只是长期观察
- 候选机会
- 接近可以研究建仓
- 已经进入重点跟踪
- 当前更接近：
  - 战略观察
  - 战术候选
  - 执行载体

## 7. 最重要的一句话

- 这家公司当前最值得继续盯的一件事是什么？
