# 下一周检查

目的：

- 不把精力耗在天天盯盘
- 在局势变化不快时，一周做一次更高层的检查就够

## 一、这周要先回答什么

不是问：

- 下周涨还是跌

而是问：

- 下周值不值得我提高关注等级？
- 下周有没有需要提前留意的重要窗口？
- 当前更应该用短期、阶段还是长周期视角？

## 二、每周只看 4 件事

### 1. 下周有没有重要窗口

例如：

- 国新办/政治局/央行/证监会重要会议或发布会
- 财报、政策窗口、利率决议
- 已知的高敏感国际事件

### 2. 这周市场在交易什么

只抓主线，不求全：

- 风险升级
- 政策预期
- 情绪修复
- 基本面变化

这一项可以再压成一句固定问题：

- `当前市场是在继续给最坏情景加价，还是已经开始给“没那么坏”加价？`

如果是战争、地缘或重大风险事件周，这一句尤其重要。

### 3. 当前最适合的时间尺度是什么

问自己：

- 现在更像短期事件交易？
- 更像阶段交易？
- 还是已经该切到长周期判断？

### 4. 下周我需要投入多少精力

分三档：

- `普通跟踪`
- `提高注意力`
- `建议重点留意`

## 三、如果本周在研究黄金，只额外补看 4 件事

这部分不是每周都必须看。

只有当你当前想判断“黄金这类风险暴露值不值得研究或参与”时，再补这一段。

### 1. 金价这周本身是走强、震荡，还是高位回撤

先不急着解释原因，先看价格行为本身。

### 2. 美元这周有没有继续明显走强

因为美元偏强时，黄金经常会受压。

### 3. 利率压力有没有继续上修

不用一上来盯很复杂的实际利率，先粗看：

- 美债收益率有没有继续明显上行
- 市场是不是更在担心“高利率更久”

### 4. 地缘 / 央行买金背景是在强化还是弱化

这一步不求每天跟，只问：

- 本周是不是又出现了新的地缘升级
- 央行买金、多元储备、货币担忧这些长期支撑有没有被明显削弱

一句话：

- 黄金每周先看：`金价 + 美元 + 利率 + 地缘/央行买金背景`
- 然后回答：它现在更像在交易避险，还是在被美元和利率压着

## 四、当前更适合你的使用方式

如果局势不快：

- 一周做一次 `下一周检查`
- 平时只保留最轻的日常关注

如果局势突然变快：

- 再切回晚间检查或预警模式

## 五、触发词

周末或周日晚上发一句：

```text
按下一周检查帮我看一下：
1. 下周有没有重要窗口
2. 当前市场主线是什么
3. 当前更适合哪个时间尺度
4. 下周我应该用普通跟踪、提高注意力，还是建议重点留意
```
