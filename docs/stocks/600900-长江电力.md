# 长江电力 `600900`

## 1. 当前定位

- 角色：`权益防守仓 / 高现金流 / 低波动股息型核心资产`
- 当前状态：`已买入 1.5%，小型配置仓已建立`
- 更适合的动作定义：`1% - 1.5% 配置型小仓`

## 2. 为什么进入系统

这张卡进入系统，不是因为它会成为最强弹性票，
而是因为它更接近：

- `高股息 + 大体量 + 强现金流` 的低波动权益资产
- 当你想提高 A 股参与度，但又不想把新增预算都押在高弹性板块时的缓冲层
- 在组合里承担 `稳住净值波动` 与 `保留权益参与` 的双重任务

如果当前更强判断是：
`中国 / A 股整体值得参与，但不想把所有新增暴露都压到第二波弹性票`
那么长江电力是合适的候选对象。

## 3. 公司定位

按公司 2024 年年报：

- 长江电力是国内核心大型水电运营平台
- 业务以 `境内水电` 为主，同时布局部分其他清洁能源与相关投资
- 核心特征不是高弹性，而是 `稳定发电资产 + 强现金流 + 持续分红能力`

核心理解：
它更像 `权益里的低波动防守资产`，
而不是：
`市场修复第二波里的高进攻性承接者`

## 4. 业务与利润来源

按 2024 年年报：

- `2024 年营业收入 844.92 亿元，同比 +8.12%`
- `2024 年归母净利润 324.96 亿元，同比 +19.28%`
- `2024 年扣非归母净利润 325.08 亿元，同比 +18.17%`
- `2024 年经营活动现金流净额 596.48 亿元，同比 -7.88%`

主营结构：

- `境内水电收入 744.79 亿元，同比 +7.87%`
- `其他行业收入 98.06 亿元，同比 +11.10%`

利润分配预案：

- `2024 年每 10 股派发现金股利 9.43 元`
- 按年末总股本计算，全年拟分派现金股利约 `230.74 亿元`

当前理解：

- 这是一家靠 `发电资产` 和 `现金流能力` 吃饭的公司
- 投资它的核心理由不是“业绩突然爆发”
- 而是“稳定盈利、稳定分红、相对抗波动”

## 5. 当前价格位置

以 `2026-04-14` 收盘和 `2026-03-02` 至 `2026-04-14` 的历史日线为准：

- `最新收盘价 26.42 元`
- 近一个多月大致运行在 `26.12 - 27.63` 区间
- 近期是 `缓慢回落后横住`，不是明显加速主升

当前理解：

- 它不是情绪高位脉冲票
- 但也不是深度低估、没人发现的低位错价
- 更像 `机构长期偏好的稳定资产`

## 6. 为什么它和当前组合有关

它与“高摩擦世界”的关系，不是直接受益于战争，
而是更偏：

- `能源安全`
- `大型清洁电源`
- `高股息防守`
- `类债型权益替代`

放进当前组合，更合理的意义是：

- 当你想增加权益参与，但又不想只买高弹性方向时，
  给组合加一层更稳的权益底座

所以它在组合里解决的问题是：

`不是帮你抓最强第二波，而是防止你为了参与第二波，把新增权益全做成高波动暴露。`

## 7. 关键验证信号

后续继续跟踪，优先看：

1. 发电量和来水情况是否继续支撑盈利稳定
2. 分红政策是否继续保持稳定
3. 经营现金流是否维持强势
4. 市场是否继续给高股息低波动资产合理溢价
5. 它是否保持“跌得慢、回撤小”的组合稳定器属性

## 8. 主要风险与反方意见

- `水文风险`：来水不及预期，会影响发电量和盈利预期
- `防守拥挤风险`：若市场风格转向高弹性，它可能相对跑输
- `类债资产重定价风险`：若高股息资产整体吸引力下降，估值会被压
- 它不是低估值黑马，不能期待高斜率超额收益
- 它最大的常见问题不是大跌，而是：
  `很稳，但涨得不够快`

## 9. 当前结论

当前更适合把长江电力定义成：

`在提高 A 股参与度时，用来压组合波动的小型配置仓`

所以更贴近系统的动作不是：

- `把它当第二波主攻票`
- `因为它很稳，就无限放大仓位`

而是：

- `把它当 1% - 1.5% 的防守型权益仓`
- `如果你想加的是稳定性，可以优先加它`
- `如果你想加的是第二波弹性，不应优先加它`

一句话说：

`长江电力适合在组合里解决“怎么更稳地参与权益”，不适合单独承担“帮你抓住第二波弹性”的任务。`
