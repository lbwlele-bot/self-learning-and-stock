# 瑞芯微 `603893`

## 1. 当前定位

- 角色：`公司层重点观察卡`
- 当前状态：`高 conviction 观察对象，已形成 1% 判断参与规则，暂未执行`
- 所属方向：`端侧 AI / AIoT / 边缘算力平台 / 中国现实 AI 路线`

## 2. 为什么进入系统

这张卡进入系统，不是因为它是最热 AI 龙头，
而是因为它更接近下面这类对象：

- 真实产品已经在市场上形成竞争力
- 受益路径不只停留在模型叙事，而能落到芯片平台、客户导入和出货
- 有机会从 `现平台兑现` 走到 `下一代平台抬高天花板`
- 用户本人在 `NXP i.MX` 路线上的工作经验，能提供比普通投资者更强的竞品理解

## 3. 公司定位

当前更合适的理解是：

`具备多代平台能力、已在端侧 AI / AIoT 场景中持续兑现，但仍处于“平台化成长”验证中的中国 SoC 公司`

它不是纯“大模型”映射，
也不只是普通消费电子周期股。

按公司 2024 年报与 2025 年业绩预告披露，
它已经把增长重心逐步推向：

- `AIoT`
- `汽车电子`
- `机器视觉`
- `工业应用`
- `机器人`

## 4. 当前最重要的研究优势

对这家公司，用户真正的优势不在于“读懂年报”，
而在于更早识别下面几件事到底是真是假：

1. `RK3588` 的现实竞争力是否仍在
2. 下一代旗舰是否真的在推进，而不是只停留在故事层
3. Fabless 路线下，设计、tapeout、产能、封装交付是否顺畅
4. 新一代平台是否真的对应更高价值场景，而不只是参数升级
5. 汽车市场突破到底是设计导入，还是概念表达

这是系统里少有的“用户具备一手行业体感”的对象。

## 5. 业务与利润来源

按公司 2024 年报与 2025 年度业绩预告：

- `2024 年营收 31.36 亿元，同比 +46.94%`
- `2024 年归母净利润 5.95 亿元，同比 +341.01%`
- `2025 年营收预计 43.87 - 44.27 亿元`
- `2025 年归母净利润预计 10.23 - 11.03 亿元，同比 +71.97% - 85.42%`

当前理解：

- 收入和利润不只是来自泛复苏
- 现平台 `RK3588 / RK3576 / RV11` 已经是主要兑现抓手
- 但增长里仍混有 `行业复苏 + 平台放量 + AIoT 渗透` 三重因素

所以更稳的定义是：

`AIoT 渗透已经开始兑现，但暂时还不是“纯 AI 收入股”。`

## 6. 当前市场位置

以 `2026-04-09` 的公开行情为准：

- 盘中/下午价格约 `160.31 元`
- 近 60 个交易日高点约 `209.51 元`
- 近 60 个交易日低点约 `145.45 元`

当前理解：

- 它已经不是无人关注的低位票
- 但也没有重新回到最乐观预期完全打满的位置
- 最近一段的回撤，较大程度也受全 A 与 AI 板块整体风险偏好影响

## 7. 当前核心工作假设

1. `现平台兑现仍在继续`
即使暂时不押注下一代旗舰，`RK3588 / RK3576` 仍可能继续支撑增长。

2. `下一代平台决定天花板，但不必作为第一笔参与的唯一触发点`
`tapeout / 产能 / 设计导入` 更适合作为后续加仓与升级确认点，
不一定非得等它们全部确认后才允许第一笔动作。

3. `如果市场因为系统性波动给出回撤，而主逻辑未坏，这类位置比“全确认后追高”更适合参与`

## 8. 第一笔参与与分级加仓规则

这条规则的目标不是“证明自己必须马上买”，
而是把 `高认知`、`高置信` 和 `怕踏空` 分开，
并允许在证据升级后，把原本过于僵硬的单点价格规则，
升级成更适合高 conviction 标的的分级参与规则。

### 当前已确认的个人执行计划

- 用户已明确：
  `瑞芯微值得作为可升级到中高仓位的高 conviction 标的，而不只是一笔永远停留在 1% 的观察仓`
- 这笔仓位的角色分三层：
  - `第一层`：`判断参与仓 + 职业/行业暴露对冲仓`
  - `第二层`：`高 conviction 个股研究仓`
  - `第三层`：`在主逻辑持续兑现前提下的核心成长仓`
- 当前已确认的个人执行理由：
  - `与本人工作路线构成强竞争，持有对手公司股票可形成一定职业暴露对冲`
  - `基于行业体感，看好公司未来发展、市场渗透与客户认可度`
  - `现平台兑现正在发生，且 2026Q1 预告进一步强化了这一点`
  - `下一代平台决定天花板，但不必成为仓位从 1% 升级到更高水平的唯一前提`
- 当前已确认的持有态度：
  - `不因为错过 150 而补偿性追高`
  - `也不把 150 以下当成唯一合法入口`
  - `先承认自己对公司有高权重输入，再用分级动作去承接，而不是要么全等、要么全追`
- 当前不接受的执行方式：
  - 放量长阳追入
  - 因踏空焦虑在板块高潮日临时上大仓位
  - 在没有明确回踩、横盘承接或市场降温的情况下，一步切成重仓

### 可参与的前提

- 自己承认这是一只 `高 conviction 标的`，但任何一次加仓仍然要有 `位置理由`
- 买入理由必须以 `现平台仍在兑现 + 长期平台化潜力 + 自身行业优势输入` 为主
- 不能把“错过 150 左右”当作主要买入理由
- 不能把“科技股涨得很猛”当作仓位升级的主要理由

### 第一笔触发方式

- 基础动作：`0.5% - 1%`
- 时点：优先在 `回撤 / 横盘 / 系统性波动给出的位置` 参与
- 原先的 `150 元以下` 规则继续保留为 `最舒服的理想区`，
  但不再是唯一允许出手的价格条件
- 在 `2026Q1 预告` 已显著强化证据链后，
  如果股价不再回到 150 以下，但出现：
  - `预告后回踩不破结构`
  - `放量上涨后的横盘承接`
  - `板块降温而个股不明显转弱`
  也允许执行第一笔 `0.5% - 1%` 判断参与仓
- 不在 `放量长阳、情绪明显高潮` 时追
- 不强行绑定 `tapeout` 单点确认

### 仓位升级框架

这里不再要求“必须等正式季报和下一代平台兑现后才能谈升级”，
但也不接受“因为长期看好，所以直接一步重仓”。

#### 第一层：`0.5% - 1%`

适用：
- 方向判断已成形
- 2026Q1 预告已提供新增证据
- 位置不是最舒服，但也不是情绪高潮追价

目标：
- 建立真实连接
- 避免持续制造机会亏损
- 让后续加仓有基准成本和观察坐标

#### 第二层：`2% - 4%`

只要满足下面任一组，就允许从第一层升级，而不必死等正式季报或下一代平台完全兑现：

1. `现平台兑现强化组`
   - `RK3588 / RK3576 / RV11` 的增长继续被公司口径、渠道体感或市场表现验证
   - 个股在板块波动中仍表现出相对承接，而不是纯情绪跟涨
   - 股价经过一次回踩或横盘后，没有明显跌回“预告前弱势状态”

2. `位置再给机会组`
   - 没有新的主逻辑破坏
   - 个股出现 `非恐慌型回撤`
   - 回撤更多像板块降温，而不是公司逻辑转坏

#### 第三层：`5% - 8%`

允许继续升级到更高仓位，但必须看到“高 conviction 不再只来自你自己，也开始被市场和公司层共同验证”：

- 现平台兑现继续推进，且没有明显边际转弱
- 公司持续给出更具体的产品、客户、量产、生态信号
- 个股不再只是题材行情中的弹性票，而开始呈现独立强势或独立承接
- 加仓位置依然来自 `回踩 / 横盘 / 分歧`，而不是情绪高潮日

#### 超过 `8%` 的前提

超过 `8%` 以后，就不再是“普通高 conviction 个股”，
而是在把它当成组合核心仓对待。

只有当下面三件事同时更清楚时，才讨论是否继续上提：
- 你自己的行业优势判断仍然稳定
- 公司兑现不只是停留在预告和故事层
- 仓位升级不是为了补偿性追赶，而是为了提高组合表达质量

### 不补与反证条件

如果后面出现下面组合，就不自动补：

- 下一代延期预期显著强化
- 现平台增速放缓
- 汽车/工业等新场景没有继续推进
- 个股弱于板块，且不是单纯系统性波动

## 11. 主要风险与反方意见

- 它已经被市场充分看见，不能假装是无人区
- 2025 年高增长里，混有明显的周期修复和低基数因素
- 现平台如果放缓，而下一代又延迟，估值会很尴尬
- 汽车故事如果迟迟不转成设计导入，会削弱长期想象空间
- 如果买入理由变成“我后悔没在 150 买”，那就是错误理由

## 12. 当前结论

当前更合适的定义是：

`可以从轻仓判断参与开始、并在证据与位置继续配合时，逐级升级到中高仓位的高 conviction 观察对象`

它适合：

- 作为 AI 个股观察主线里的第一优先对象
- 作为“专业理解能否转化成投资优势”的重点样本
- 在系统性波动给出位置、而主逻辑未坏时，先建立 `0.5% - 1%` 级别参与
- 在现平台兑现继续推进、且个股承接没有明显破坏时，允许升级到 `2% - 4%`
- 在公司与市场共同验证强化后，再讨论是否提升到更高核心仓位

它暂时不适合：

- 因为错过低位而补偿性追高
- 在板块高潮日因为怕踏空而直接一步上大仓位
