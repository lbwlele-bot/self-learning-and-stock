# 阶段摘要

## 当前阶段

当前系统处在：
`投资系统分层清晰化 + 发现路径能力建设`

重点不是继续扩主题，而是让：

- 主题导航更清楚
- 逻辑基点归属更稳定
- 研究对象发现路径更可复用
- 执行层不再和学习层混写

## 当前状态

- 学习系统已嵌入投资系统前半段
- 逻辑基点已初步成形
- 工作假设正在持续扩展
- 发现路径已建立，但热度过滤能力仍在强化
- 短中期落地仍以黄金、ETF、板块等宏观表达为主

## 本轮新增收束

- `高摩擦世界` 更接近长期危机化运行，而不是一次性硬脱钩
- `美元武器化` 短中期更可能先抬升全球风险溢价与流动性分层，再反映为中国资产折价常态化
- A 股更值得关注的不是整体叙事反转，而是整体折价下的局部承接者
- 散户更适合让宏观负责缩小搜索范围，让公司层验证决定是否继续研究或下注
- 执行层已补入 `分层吸纳`：对 `宽基 / ETF / 资产配置型标的 / 高 conviction 核心资产`，不再一律只允许“等确认后追”，但必须先写明总预算、分层比例与停止条件
- 系统开始补 `组合预算与角色分工`：当更强判断是“中国 / A股整体值得参与”而不是“已高度确认某只个股会明显跑赢”时，预算默认先给 `宽基 / ETF`，再给 `主题表达`，最后才提高 `高 conviction 个股`
- 执行层开始补 `退出规则`：先围绕当前真实持仓，把 `继续持有 / 冻结加仓 / 兑现一部分 / 降级持有 / 全部撤回` 统一成标准动作语言，并明确不同角色不能用同一把尺子处理
- 系统开始补 `估值带 -> 动作带`：把 `便宜但未稳 / 便宜且逻辑未坏 / 合理区 / 偏贵但逻辑继续强化 / 昂贵且乐观打满` 这些状态翻成 `继续吸纳 / 冻结加仓 / 兑现一部分 / 降级持有` 的动作偏向
- `AI` 主题应从“技术/产业主线”升级为“技术 + 制度 + 分配 + 治理”的复合主题
- 高摩擦的外延正在从传统金融与地缘风险，扩展到智能基础设施、审计标准、模型接入与国际治理接口
- 白银在系统里初步定位为“贵金属方向上的高弹性混合暴露”，角色上后于现金与黄金，不作为核心安全资产

## 当前最需要防止的事

- 文件越来越多但职责越来越混
- 同一个概念在多个总览文件重复重讲
- 把类型样本误当成预期差研究对象
- 把学习讨论直接跳成执行动作

## 当前一句话

系统正在从“会讲逻辑”继续推进到“逻辑能找到对象、对象能进入管理、管理再进入执行，并开始补齐退出与持仓管理”。
