# Repo-Local Skills

This directory is the workflow entry layer for the repository.

It does not replace:

- `START_HERE.md`
- `SYSTEM_INDEX.md`
- `TOPIC_MAP.md`
- `memory/`

Use the current split like this:

- root files plus `memory/` are the truth layer
- `docs/` is the method reference layer
- `skills/` is the workflow entry layer

## Default Chain

Run the current topic through:

`system-resume -> discovery-pipeline -> company-card-builder -> market-ops -> consent-capture -> memory-governor`

Use `consent-capture` only when part of the conversation should be promoted into the long-term system and the user should approve the write first.

## Current Skills

- `system-resume`
  Resume the active system state, topic, assumptions, and next workflow
- `memory-governor`
  Route a new conclusion into one primary file or artifact
- `market-ops`
  Run daily, alert, nightly, next-day, and weekly market workflows
- `retro-replay`
  Run historical replay without leaking future information
- `discovery-pipeline`
  Turn a topic into industries, key links, heat filtering, and watchlist candidates
- `company-card-builder`
  Build or refresh a company research card from a vetted candidate
- `consent-capture`
  Distill reusable conversation insights, propose storage targets, and wait for explicit approval before writing

## Current Mainline

For the current mainline around:

- high-friction world

start with:

1. `system-resume`
2. `discovery-pipeline`

Only move to:

4. `market-ops`
